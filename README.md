# PDM Dealership

PDM MLO developed by [Patoche](https://www.gta5-mods.com/users/Patoche) which has been [released for free](https://www.gta5-mods.com/maps/car-dealer-fivem-sp-ready).

All the credit goes to the original creator, Patoche. Since this map is being used in ESX-Overextended, its files needed to be uploaded on GitHub for the [txAdmin recipe](https://github.com/esx-overextended/txAdminRecipe) to be able to download it automatically.
